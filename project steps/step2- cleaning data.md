# Ferns and Petals Sales Analysis

## Step 2: Data Cleaning

### Project Overview
This project analyzes the sales data of **Ferns and Petals (FNP)** to understand:
- Customer behavior
- Sales trends
- Product performance
- Delivery efficiency
- Occasion-based sales

After extracting the datasets into Excel using **Power Query Editor (PQE)**, the next step is **Data Cleaning**.

Data cleaning ensures that the dataset is:
- Accurate
- Consistent
- Error-free
- Ready for transformation and dashboard analysis

---

# Purpose of Data Cleaning

The main objective of data cleaning is to:
- Remove null values
- Detect errors
- Identify duplicates
- Correct inconsistent data
- Verify data quality
- Prepare the dataset for analysis

---

# Open Power Query Editor (PQE)

After importing the datasets:

1. Go to:
   **Data → Queries & Connections**
2. Open the imported query
3. Click:
   **Transform Data**

Now the dataset opens inside **Power Query Editor (PQE)**.

---

# Method 1: Using Column Profiling

## Steps

1. Select all columns
2. Go to:

   **View → Column Profiling**

3. Choose one option:
   - Based on Top 1000 Rows
   OR
   - Entire Dataset

---

# What Column Profiling Analyzes

Column Profiling helps identify:
- Null values
- Error values
- Unique values
- Duplicate values
- Data distribution
- Statistical information

---

# Method 2: Enable Data Quality Features

## Steps

Go to:

**View → Enable Checkboxes**

Enable the following options:

- Column Quality
- Column Distribution
- Column Profile

---

# Features and Their Purpose

## 1. Column Quality
Used to check:
- Valid values
- Empty values
- Error values

---

## 2. Column Distribution
Used to analyze:
- Distinct values
- Duplicate records
- Frequency distribution
- Unique records

---

## 3. Column Profile
Used to view:
- Minimum value
- Maximum value
- Average value
- Count statistics
- Data patterns
